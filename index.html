D3.Geo Cookbook

This page consists of basic, reduced mapping examples using the D3 javascript visualization library. The primary reason this cookbook exists is due to a consistently growing need for example-based learning. More often than not, people will run into brick walls during their learning of D3 because of a lack of simplified examples.

There are an abundance of projects using D3 existing on the web, but many of them are quite complex and do not allow the average and curious learner to simply learn specific components of the projects.

This D3 cookbook attempts to collect stand-alone examples using the d3.geo library so people can see how specific map tasks are completed without anything else getting in the way.
